## Gradle Commands

### Create a new project

```shell
mkdir project
cd project
gradle init
```

### List available Gradle tasks

```shell
gradle task
```

### Run Gradle tests

```shell
 gradle test --info
```

### Create a distribution zip with dependencies

```shell
gradle distZip
```
